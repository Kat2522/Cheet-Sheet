Headings

Headings go from h1 to h6. All html documents should start with a h1. This is because h1 is used by search engines and there should only be one h1. Note: do not use h1-6 to format text size!

Paragraphs

HTML paragraphs can be any structural grouping of related content, such as images or form fields. p is a block level element.
