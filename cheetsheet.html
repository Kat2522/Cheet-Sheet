Headings

Headings go from h1 to h6. All html documents should start with a h1. This is because h1 is used by search engines and there should only be one h1. Note: do not use h1-6 to format text size!

Paragraphs

HTML paragraphs can be any structural grouping of related content, such as images or form fields. p is a block level element.

Lists

Lists are a way of organizing information, each bit of information uses li to encapsulate it. The whole list is then formatted using the type of list desired encapsulating all li elements.

Organizing Content

The 'div', 'span', and 'br' elements are all useful for grouping similar elements together, and for separating one group of content from another.

Div is a block type element so it will show up on an new line...
